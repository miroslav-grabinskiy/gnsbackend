# How to start
1. install `npm` latest,
2. install, run `mysql`
3. set settings to `/config/config.json`
4. open directory project in `terminal`
5. `npm install`
6. `npm i sequelize -g`
`npm i myqsl2 -g`
`sequelize db:migrate` or run script from `/migrations/first.txt` in mysql manually
7. `npm run reg_admin`
8. `npm start`
